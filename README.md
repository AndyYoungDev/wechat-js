# 如何使用
通过composer安装yii2  <br>
`composer create-project --prefer-dist yiisoft/yii2-app-basic wechat`  <br>
复制文件到项目中  <br>
修改配置文件params.php  <br>
配置文件中参数wechat[appId],wechat[appSecret],wechat[token]
